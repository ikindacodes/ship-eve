# Integrations

Connect TaskFlow to the tools your team already uses.

## Slack

To connect Slack:

1. Open **Settings → Integrations**.
2. Find **Slack** and click **Connect**.
3. Choose the Slack workspace and channel for notifications.
4. Click **Allow**.

TaskFlow posts task updates and @mentions to the selected channel.

## GitHub

To link a GitHub repository:

1. Open **Settings → Integrations**.
2. Click **GitHub → Connect**.
3. Select the organization and repository.
4. Enable **Sync pull requests** if you want PR status on TaskFlow cards.

Disconnect anytime from the same Integrations page.

## Webhooks

Enterprise workspaces can configure outbound webhooks under **Settings → Integrations → Webhooks**.

Pro and Free plans do not include custom webhooks.
